

# Типы стек(stack) и куча(heap)

## Стек(stack) 1 мб. - быстро

[![image.png](https://i.postimg.cc/HkwyWKbk/image.png)](https://postimg.cc/9DfMJLx6)

Используется для хранения переменных, используется для вызова методов. (Выполняет логику программы), работает по принципу "Last in first out".

## Куча(heap) 32х 1.5 гб. 64х 8тб. - медленно
[![image.png](https://i.postimg.cc/Zn9hjGJP/image.png)](https://postimg.cc/fShFRgtk)

Накладно выделять память в куче. От неё проблематично очистить оперативную память. Используют сборщик мусора (иначе будет утечка памяти).
